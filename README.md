# Weather App

**Description:**  
A simple web app that fetches real-time weather data using the OpenWeatherMap API and displays it in a clean user interface.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- OpenWeatherMap API

## Features
- Real-time weather data
- Search weather by city name
- Responsive design
- Humidity
